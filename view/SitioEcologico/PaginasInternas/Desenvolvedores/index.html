Sobre o Projeto

Este WebGIS é um projeto desenvolvido e mantido por alunos da 18ª turma do curso de Geoprocessamento da Fatec Jacareí, como parte de um trabalho acadêmico que visa aplicar conhecimentos em geotecnologias, sistemas de informações geográficas e desenvolvimento web.

A plataforma foi criada para fornecer uma ferramenta de fácil acesso e interação para a visualização de dados espaciais, contribuindo para a análise e compreensão de fenômenos geográficos.
